# Odin Landing Page

## a brief introduction

This repo is an exercise about a brief and training for basics of css by making a landing page.

## Skills Demonstrated

- **UI/UX Replication & Theming:** Successfully analyzed a wireframe layout and translated it into a fully themed, customized restaurant landing page design.
- **Advanced Layout Design:** Leverage CSS Flexbox positioning to construct responsive, aligned grid elements, content blocks, and navigation bars.
- **Typography & Visual Hierarchy:** Applied specific styling rules, font variations, and high-contrast color choices to emphasize readability and user focus.
- **Asset Management:** Handled structured file directories by integrating custom imagery, local assets, and tailored vector icons.
- **Version Control:** Practiced disciplined Git workflow with regular, descriptive commits documenting style changes and structural adjustments.
